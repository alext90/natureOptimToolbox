# Nature Optim[ization] Toolbox
A collection of nature inspired optimization algorithms.  

Implemented so far:  
- Artificial Bee Colony  


Setup venv:  
```
make setup
```

Install requirements:  
```
make install
```

Run an example script:  
```
make run
```

Test:  
```
make test
```
